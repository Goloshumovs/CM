# Решение уравнений параболического типа

## Описание

Программа решает начально-краевую задачу для дифференциального уравнения параболического типа с использованием трёх численных методов: явной конечно-разностной схемы, неявной конечно-разностной схемы и схемы Кранка-Николсона.

## Уравнение

Решается уравнение теплопроводности вида:

$$\frac{\partial u}{\partial t} = a^2 \frac{\partial^2 u}{\partial x^2}$$

с начальными и граничными условиями согласно варианту 4.

## Методы решения

1. **Явная конечно-разностная схема** - простейший метод, но условно устойчивый
2. **Неявная конечно-разностная схема** - абсолютно устойчивый метод
3. **Схема Кранка-Николсона** - метод второго порядка точности по времени

## Аппроксимация граничных условий

Программа поддерживает три варианта аппроксимации граничных условий, содержащих производные:
- Двухточечная аппроксимация с первым порядком
- Трёхточечная аппроксимация со вторым порядком
- Двухточечная аппроксимация со вторым порядком

## Входные параметры

- a = 0.4 - коэффициент температуропроводности
- L = π - длина области
- T = 1.0 - конечное время
- Nx = 50 - количество узлов по пространству
- dt = 0.0005 - шаг по времени

## Выходные данные

- График сравнения численного и аналитического решений в различные моменты времени
- График ошибки численного решения
- Максимальная ошибка решения

## Пример запуска

# Выбор метода
1 - явная конечно-разностная схема
2 - неявная конечно-разностная схема
3 - схема Кранка-Николсона

# Выбор аппроксимации
1 - двухточечная аппроксимация с первым порядком
2 - трёхточечная аппроксимация со вторым порядком
3 - двухточечная аппроксимация со вторым порядком

## Результаты

Схема Кранка-Николсона показывает наилучшую точность среди всех методов. При увеличении шага по координате x погрешность вычислений увеличивается.

# Решение уравнений гиперболического типа

## Описание

Программа решает начально-краевую задачу для дифференциального уравнения гиперболического типа (волновое уравнение) с использованием явной схемы крест и неявной схемы.

## Уравнение

Решается волновое уравнение вида:

$$\frac{\partial^2 u}{\partial t^2} = a^2 \frac{\partial^2 u}{\partial x^2} + cu$$

с начальными и граничными условиями согласно варианту 4.

## Методы решения

1. **Явная схема крест** - условно устойчивый метод
2. **Неявная схема** - абсолютно устойчивый метод

## Аппроксимация граничных условий

Программа поддерживает три варианта аппроксимации граничных условий:
- Двухточечная аппроксимация с первым порядком
- Трёхточечная аппроксимация со вторым порядком
- Двухточечная аппроксимация со вторым порядком

## Аппроксимация второго начального условия

- Первый порядок точности
- Второй порядок точности

## Входные параметры

- a = 1 - скорость распространения
- c = -5 - коэффициент
- x0 = 0, xl = 1 - границы области
- t = 2 - конечное время
- m = 100 - количество узлов по пространству
- n = 1000 - количество узлов по времени

## Выходные данные

- График сравнения численного и аналитического решений в моменты времени t = 0, 1, 2
- График ошибки численного решения
- Максимальная погрешность

## Пример запуска

# Выбор метода
1 - явная схема крест
2 - неявная схема

# Выбор аппроксимации
1 - двухточечная аппроксимация с первым порядком
2 - трёхточечная аппроксимация со вторым порядком
3 - двухточечная аппроксимация со вторым порядком

# Выбор порядка аппроксимации второго начального условия
1 - первый
2 - второй


## Результаты

Оба метода успешно решают задачу. При выборе явной схемы необходимо проверять условие устойчивости. Неявная схема является безусловно устойчивой и даёт хорошие результаты при любых шагах сетки.
