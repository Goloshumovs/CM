# Решение уравнений параболического типа

## Описание

Программа решает начально-краевую задачу для дифференциального уравнения параболического типа с использованием трёх численных методов: явной конечно-разностной схемы, неявной конечно-разностной схемы и схемы Кранка-Николсона.

## Уравнение

Решается уравнение теплопроводности вида:

$$\frac{\partial u}{\partial t} = a^2 \frac{\partial^2 u}{\partial x^2}$$

с начальными и граничными условиями согласно варианту 4.

## Методы решения

1. **Явная конечно-разностная схема** - простейший метод, но условно устойчивый
2. **Неявная конечно-разностная схема** - абсолютно устойчивый метод
3. **Схема Кранка-Николсона** - метод второго порядка точности по времени

## Аппроксимация граничных условий

Программа поддерживает три варианта аппроксимации граничных условий, содержащих производные:
- Двухточечная аппроксимация с первым порядком
- Трёхточечная аппроксимация со вторым порядком
- Двухточечная аппроксимация со вторым порядком

## Входные параметры

- a = 0.4 - коэффициент температуропроводности
- L = π - длина области
- T = 1.0 - конечное время
- Nx = 50 - количество узлов по пространству
- dt = 0.0005 - шаг по времени

## Выходные данные

- График сравнения численного и аналитического решений в различные моменты времени
- График ошибки численного решения
- Максимальная ошибка решения

# Курсовой проект: Вычисление многократных интегралов с использованием квадратурных формул и метода Монте-Карло

## Описание проекта

Программа предназначена для численного вычисления одномерных, двумерных и трёхмерных интегралов с использованием двух подходов: метода Монте-Карло и квадратурных формул Гаусса. Разработана на языке Python с графическим интерфейсом на Tkinter.

## Актуальность

Численное интегрирование является важной областью численных методов и применяется в:
- **Физике и инженерии** - вычисление площадей, объёмов, центров масс
- **Экономике и финансах** - оценка финансовых моделей и опционов
- **Научных исследованиях** - моделирование процессов в биологии и химии
- **Компьютерной графике** - расчёт освещения и теней
- **Обработке сигналов** - анализ спектров и фильтрация

## Методы решения

### Метод Монте-Карло

Геометрический метод, основанный на случайной выборке точек:

1. Ограничим функцию прямоугольником (n-мерным параллелепипедом), площадь которого можно легко вычислить
2. «Набросаем» в этот прямоугольник N точек со случайными координатами
3. Подсчитаем количество точек K, которые попадут под график функции
4. Площадь области вычисляется по формуле:

$$S = S_{par} \cdot \frac{K}{N}$$

### Квадратурные формулы Гаусса

В общем виде формула численного интегрирования записывается следующим образом:

$$\int_{a}^{b} f(x)dx = \sum_{i=1}^{m_g} \widetilde{w_i} f(x_i) = \sum_{i=1}^{m_g} w_i \det(J(\xi_i)) f(x(\xi_i))$$

где:
- $f(x)$ — подынтегральная функция
- $w_i$ — веса интегрирования
- $\xi$ — система координат мастер-элемента
- $J(\xi)$ — матрица Якоби для перехода на мастер-элемент

$$J(\xi) = \frac{\partial(x_1, \ldots, x_n)}{\partial(\xi_1, \ldots, \xi_n)}$$

## Одномерный интеграл

### Параметры метода

- Область интегрирования: отрезок $[x_0, x_1]$
- Мастер-элемент: отрезок $[-1, 1]$

**Переход на мастер-элемент:**

$$\xi(x) = 2\frac{x - x_0}{x_1 - x_0} - 1$$

**Переход с мастер-элемента:**

$$x(\xi) = \frac{(x_1 - x_0)(\xi + 1)}{2} + x_0$$

**Якобиан:**

$$\det(J(\xi)) = \frac{x_1 - x_0}{2}$$

## Двумерный интеграл

### Параметры метода

- Область интегрирования: прямоугольник $[x_0, x_1] \times [y_0, y_1]$
- Мастер-элемент: квадрат $[-1, 1] \times [-1, 1]$

**Переход на мастер-элемент:**

$$\xi(x,y) = 2\frac{x - x_0}{x_1 - x_0} - 1$$

$$\eta(x,y) = 2\frac{y - y_0}{y_1 - y_0} - 1$$

**Переход с мастер-элемента:**

$$x(\xi, \eta) = \frac{(x_1 - x_0)(\xi + 1)}{2} + x_0$$

$$y(\xi, \eta) = \frac{(y_1 - y_0)(\eta + 1)}{2} + y_0$$

**Якобиан:**

$$\det(J(\xi)) = \frac{(x_1 - x_0)(y_1 - y_0)}{4}$$

## Вывод

Решение с помощью квадратурных формул Гаусса имеет высокий уровень аппроксимации, позволяющий получить численное решение, очень близкое к аналитическому. Однако при рассмотрении многократных интегралов с большим количеством переменных или функций, не имеющих разложения в степенной ряд или имеющих малый радиус сходимости в некоторых точках отрезка интегрирования для достижения требуемой точности интегрирования может потребовать слишком большого объёма вычислений для любых квадратур.
Решение методом Монте-Карло имеет очень большую вариативность решений и для получения достаточно близкого к аналитическому решению результата требуется обработать огромное количество точек, что сильно влияет на скорость работы программы. Для малого числа измерений интегрируемой функции производительность Монте-Карло интегрирования гораздо ниже, чем производительность детерминированных методов. Тем не менее, в некоторых случаях, когда функция задана неявно, а необходимо определить область, заданную в виде сложных неравенств, стохастический метод может оказаться более предпочтительным. Вычисленное решение методом Монте-Карло сходится к аналитическому решению. Скорость сходимости зависит от области интегрирования. Чем больше отношение площади интегрирования к площади прямоугольника, ограничивающего область интегрирования, тем быстрее сходимость метода. Аналогично для объема интегрирования и параллелепипеда, ограничивающего область интегрирования.
